RedaXtor is OpenSource with MIT licence and is distributed on GitHub

RedaXtor comes with default set of plugins for Rich Text editing, Source Code, Block Background, Standalone Image and SEO meta information editing.

As a developer you can add own plugin as long as you follow up specs. You may refer to bundle sample for reference too.

 



Edit predefined images!

You can force user to modify existing image without adding/removing new ones

Useful for cases when you want strict structure or have custom image styling you don't want to touch

Edit widget code!

If you have iframes or scripts on your page, source editor is your friend





Edit block backgrounds!

You can change color or background image of blocks

You can have different image set for usage in backgrounds and content images
